在塞尔达传说荒野之息里， 武器都有耐久度，用久了就会坏，但是完成四神兽后获得的英杰武器，在毁坏之后是可以重做的。今天小编就为大家带来塞尔达传说荒野之息英杰武器的修复办法，感兴趣的朋友快来看看吧！

　　 英杰武器的修复办法

　　 光鱗の槍 / Lightscale Trident

　　去【ゾーラの里 / Zora's Domain】，找【Dento / ロースリ】，可再做一把光鳞之枪。具体位置请看下图中的小地图。

　　光鳞之枪的材料：

　　1.ゾーラの槍 / Zora's Spear

　　2.钻石 x1

　　3.打火石 x5 【火打ち / Flint】

　　佐拉之枪可以在下图位置找到，随红月刷新。

　　钻石可以找【ゾーラの里 / Zora's Domain】的【Ledo / レトーガン】，用10颗夜光石交换钻石。

　　 巨岩砕き / Boulder Breaker

　　去【Goron City / ゴロンシティ】找【ブロハン / Rohan】，可再做一把【巨岩砕き / Boulder Breaker】

　　岩碎剑的材料：

　　1.石打ち / Cobble Crusher

　　2.钻石 x1

　　3.【火打ち / Flint】×5

　　在下图位置可获得【石打ち / Cobble Crusher】，在哥隆城·炎之神兽的主线流程中来过这里，开炮破墙救哥隆小弟。

　　钻石的获得方法如前文所述。

　　 オオワシの弓 / Great Eagle Bow

　　去利德之村【Rito Village / リトの村】找【Harth / ハーツ】，可做这把弓。

　　【オオワシの弓 / Great Eagle Bow】所需材料：

　　1.ツバメの弓 / Swallow Bow

　　2.钻石 x1

　　3.柴火 ×5

　　【ツバメの弓 / Swallow Bow】在【Harth / ハーツ】的家里桌上就能拿到。

　　钻石的获得方法如上文所述。
