为了找到并查看记忆，您必须根据您的Sheikah Slate 相册中可见线索来找到位置。 画家Pikango，可以在海拉尔的各个马舍和城镇找到，他可以提供额外的线索。 他一般在城镇或者马舍距离记忆点最近的位置。 他只会在白天被找到。

到达正确的目的地后，您会发现一个发光的地方将触发记忆，为您解锁记忆。

Memory Locations

在下面你可以找到所有的Memories可以找到的地方，包括那些在主要任务期间解锁的。

所有记忆位置都可以在塞尔达地图中看到：

有关特定记忆位置的说明，请参见下文。

MEMORY #1: SUBDUED CEREMONY
  • 照片#1
  • 地点: Sacred Ground Ruins, Central Hyrule
  • Pikango线索: Woodland Stable

相关照片显示在Central Hyrule不远处的海拉尔城堡的位置。 事实上，这个位置就在城堡南部，位于被称为神圣遗址(Sacred Ground Ruins)的地方，那里有一个小水池围着一个守护者巡逻的小屋。 只要你非常安静，你应该可以到达中心的记忆点来触发回忆。

MEMORY #3: RESOLVE AND GRIEF
  • 照片#2
  • 地点: Lake Kolomo, Central Hyrule Region
  • Pikango线索: Riverside Stable, Great Hyrule Forest Region

相册中的第二张照片显示了位于海拉尔中心的湖边地带，距离大平原(Great Plateau)不远。 从高原塔向北流向大湖，沿着西岸前往，找到一个小山脊。 这个地方很大程度上没有敌人，所以你不会在去记忆点时候遇到麻烦。

MEMORY #5: ZELDA’S RESENTMENT
  • 照片#3
  • 地点: Ancient Columns, Tabantha
  • Pikango线索: Tabantha Bridge Stable

相册的第三张照片描绘了神圣的废墟与神兽Vah Medoh的圆形区域。 它位于Tabantha大桥对面，通过了马舍。 穿过桥后，你需要爬上南方的大山脊，并注意到在构建家园Moblins，直到你到达Tena Ko'sah神社，你会发现在前面的记忆点。

MEMORY #7: BLADES OF THE YIGA
  • 照片 #4
  • 地点: Kara Kara Bazaar, Gerudo Desert
  • Pikango线索: Gerudo Canyon Stable

第四张照片显示了一片沙漠绿洲，位于Gerudo镇路上的平原。 在镇前的绿洲（Kara Kara Bazaar）停下来，环顾水边，找到记忆点。

MEMORY #8: PREMONITION
  • 照片 #5
  • 地点: Eldin Canyon, Eldin
  • Pikango线索: Gordon City

第五张照片描绘了可俯瞰Great Hyrule Forest的地区，林地塔(Woodland Tower)矗立在那里。 它位于高山的远大西部边缘的Eldin峡谷，就在林地塔的东边。 您可以通过攀登到戈朗比湖西部的高峰，并通过Lizalfos战斗，找到一个高原，被小石头环绕着，可以在这俯瞰塔，记忆就在石头前面。

MEMORY #9: SILENT PRINCESS
  • 照片 #6
  • 地点: Irch Plain, Hyrule Ridge
  • Pikango线索: Serenne Stable, Great Hyrule Forest

相册中的第六张照片描绘了从西侧俯瞰海拉尔城堡的草地。 该地点是从海拉尔城堡到达Ridgeland Tower东北部的河流，就在皇家古老的实验室遗址，那里有一棵大树在水池旁边。 记忆点可以在树的前面找到。

MEMORY #11: SHELTER FROM THE STORM
  • 照片 #7
  • 地点: West Necluda, Dueling Peaks
  • Pikango线索: Highland Stable

相册中的第七张照片显示了远距离在Gerudo Highlands俯视海拉尔大桥，将照片放在东岸。 您可以在靠近东侧的湖北端，靠近Scout山的河边找到它。 寻找到一棵大树和两个小雕像在小小的山坡上，记忆位置在附近。

MEMORY #12: FATHER AND DAUGHTER
  • 照片 #8
  • 地点: Hyrule Castle, Central Hyrule
  • Pikango线索: urelin Village, Faron

这张相册中的第八张图片描绘了在更好的时间看到的海拉尔城堡内塞尔达公主房间的入口，因此很难找到记忆点。

你最好的机会是从西边潜入海拉尔城堡，并在主要城堡本身一侧的高层上寻找一个高大的栅栏北部的尖塔。

如果你可以绕过下面的守护者，那么你只能有一个飞行守护者巡逻塔需要考虑。

MEMORY #13: SLUMBERING POWER
  • 照片 #9
  • 地点: Spring of Power, Akkala
  • Pikango线索: East Akkala Stable

相册中的第九张图片显示了一个很多水的喷泉，里面有两棵树，坐落在力量之泉，它是三座神秘泉之一。

这个泉水特别是位于Akkala高地和深Akkala之间，就在北阿卡拉谷(North Akkala Valley)北部。 看看周围的大水池，你会发现在水面前面的记忆点。

MEMORY #14: TO MOUNT LANAYRU
  • 照片 #10
  • 地点: Sanidin Park Ruins, Hyrule Ridge
  • Pikango线索: Outskirt Stable

这专辑中的第十张图片描绘了一大片非常漂亮的马雕像。 在通过Hyrule Ridge，西部从中央塔的河流的路上发现雕像。 寻找通过Satori山东部的道路，找到一座大型山丘顶端的公园，那里的雕像附近有记忆点。

MEMORY #15: RETURN OF CALAMITY GANON
  • 照片 #11
  • 地点: Lanayru Road - East Gate, Necluda
  • Pikango线索: Kakariko Village

专辑中的第十一张图片描绘了一条大路上的大门。 这可以在Lanayru Heights找到。

您可以通过离开Kakariko村庄，通过上面的山丘，通过大仙女喷泉，在找到右边的道路之前，相当简单地向前走。 这一段通过长廊，那里充满了博科比林，并以一个守卫东门的Moblin结束，在那里你会发现一个大拱门的记忆点。

MEMORY #16: DESPAIR
  • 照片 #10
  • 地点: Hyrule Field, Central Hyrule
  • Pikango线索: Wetland Stable

第一张专辑中的第十二也就是最后一张照片展示了一条相当普通的森林之路。 这个地区位于Hilia河沿岸的中央海拉尔东岸。 您可以在湿地马舍下的山顶和河流中找到这个森林覆盖的北部沼泽地。 搜索森林找到记忆片。

MEMORY #17: ZELDA'S AWAKENING
  • Main Quest Memory
  • 地点: Fort Hateno, Dueling Peaks

这个记忆最初没有在Sheikah Slate的专辑中列出，并且要求你首先找到所有的塞尔达的其他记忆。 然后，您可以在Impa的墙上看到一个充满监护人的大型战场的照片。 这个位于不远处的Kakariko村庄，在通向Necluda的名叫Fort Hateno的大墙上。 看看大战场，你会发现中心附近有大水池。

找到任何一个回忆，回到卡卡里科村的Impa，她会奖励你的蓝色冠军长袍。
