塞尔达传说荒野之息作为开放世界游戏，素材收集自然是少不了的。游戏中有些需求量大，或者较难发现的素材，玩家可能苦于不知道去哪寻找。今天小编就为大家带来玩家冰之鬼分享的塞尔达传说荒野之息部分紧缺素材高效率收集方法汇总，感兴趣的朋友们快来看看吧！

　　 部分紧缺素材高效率收集方法汇总

　　 ① 疾走蘑菇 / ゴーゴーダケ / Rushroom

　　游戏中需求量第2的素材，实质收集要求最高的素材。疾走蘑菇实际上在游戏里近乎烂大街，大多数峭壁上都可见到。然而游戏中需要近百个疾走蘑菇，所以即便如此收集也还是相当麻烦的。

　　格鲁多峡谷马宿 / ゲルドキャニオン馬宿 / Gerudo Canyon Stable 旁边的岩柱上就有不少疾走蘑菇。岩柱非常高从下面可能观察不到，爬上去也很困难，最好是从上面飞过去，往下攀爬摘取。

　　海利亚大桥 / ハイリア大橋 / Bridge of Hylia 以东、德亚村废墟 / アデヤ村跡 / Deya Village Ruins 以南有个巨大的空心树桩，里面长了不少疾走蘑菇。

　　当然这点数量的蘑菇也只是杯水车薪，疾走蘑菇更重要的是平时爬山的时候尽量顺带收集，这大概是要伴随大多数人半数的荒野之息游玩时间。

　　 ②柴火 / 薪の束 / Firewood

　　柴火除了平时用来点火取暖、切换游戏内时间外，还有两个重要的支线任务需要庞大的数量，本游戏需求最大的素材。当然由于树木随处可见，大量马宿提供一些免费取用的柴火，实际收集难度并不大。

　　然而那两个支线任务开始时大多数玩家可能就跑树林里炸树了，是不是感觉非常枯燥?实际上游戏里有快速收集柴火的方法。

　　其实游戏内的利特马宿 / リトの馬宿 / Rito Stable 是个伐木场，周围有不少圆木堆。这些圆木堆不只是摆设，只要用金属武器砍两下，或者一个炸弹，就能变成柴火。

　　利特马宿周围有4堆圆木，可以获得19个柴火。而这些圆木堆在你传送后就会恢复，可以反复获取柴火。只要往复几次，任务所需的柴火就能到手。

　　 ③星之碎片 / 星のかけら / Star Fragment

　　游戏里需求量不大，但却是多个关键装备所需的素材。星之碎片入手可以说非常简单，也非常困难：晚上看到流星，在天亮之前前往流星坠落的地方拾取。

　　光柱就是流星坠落的地点

　　天亮之前没有拾取就会消失。

　　除此以外，有两个支线任务的报酬是星之碎片，分别在：

　　平原外马宿 / 平原外れの馬宿 / Outskirt Stable

　　森林马宿 / 森の馬宿 / Woodland Stable

　　有至少两个宝箱内装的是星之碎片，分别在：

　　ドイブラン遺跡 / Thyphlo Ruins

　　タルタル台地 / Tal Tal Peak 东边靠海山道

　　另外白银人马有可能掉落，和狗打好关系带你找的宝箱中也有可能获得。

　　 ④海拉尔鲈鱼 / ハイラルバス / Hyrule Bass

　　所谓的杂鱼(...)游戏里最早需要的素材之一，后续也还需要一些。有些玩家可能一开始不知道去哪里找，或者到后面忘了去哪里找了。

　　在游戏一开始的时之神殿 / 時の神殿跡 / Temple of Time 旁边的池塘里就尽是海拉尔鲈鱼，尤其是聚集在池塘中央的石柱附近。

　　 ⑥耐火蜥蜴 / ヒケシトカゲ / Fireproof Lizard

　　一个非常重要的支线任务所需要的素材，后续升级也需求不小。实际上耐火蜥蜴非常好收集，关键仅是技巧。

　　南采掘场 / 南採掘場 / Southern Mine 里耐火蜥蜴近乎随处可见，装备上忍者装即可方便的收集。除了躺地上的，南采掘场里的小石头举起来，下面基本都有耐火蜥蜴。注意的是A键举起石头后，立刻按A键就能抓到蜥蜴，不需要把石头扔掉(蜥蜴早跑了)。

　　 ⑦草莓 / イチゴ / Wildberry

　　生活在寒冷地带的植物，一个支线任务需求的素材。由于野外的草莓不太显眼，可能有些人一直没发现。

　　草莓最方便采集的地方同样是利特马宿 / リトの馬宿 / Rito Stable，周围一圈刚好10个。

　　由于利特马宿离任务地点很远，也有按照任务的方法。在格鲁多高地 / ゲルド高地 / Gerudo Hignlands 右上角的泊欧台地 / オルパー台地 / Laparoh Mesa，台地的右下角有一小块有植被覆盖的地方，这里就能采集到大量草莓。

　　 ⑧雷电蜻蜓 / ビリビリヤンマ / Thunderwing Butterfly

　　一个支线任务需要这种蜻蜓，大量玩家卡在这里。这种蜻蜓比较特殊，仅在雨天出现。由于雨天时视线不佳，不容易发现飞行的昆虫，一些人一直没能抓到甚至没能见到。

　　要刻意找雨天并在一堆雨点中找到蜻蜓并不容易，有没有简单的方法?

　　河畔马宿 / リバーサイド馬宿 / Riverside Stable 的商人泰利的商品中就有雷电蜻蜓，而且仅需要10卢比(是不是有一种你他妈逗我的感觉)。

　　任务只需要一只，花钱搞定吧。

　　 ⑨耐力蜜蜂蜂蜜 / ガンバリバチのハチミツ / Courser Bee Honey

　　支线和升级需要的素材。蜂巢同样遍布游戏内很多地方，但不少玩家平时并不会去招惹蜜蜂，等需要用到时会比较尴尬——蜂巢分布非常零散，不方便寻找与收集。下面是蜂巢相对集中的地点。

　　哈特诺要塞 / ハテノ砦 / Fort Hateno 东面有一件木屋，周围就有数个蜂巢(某人你正职养蜂的么...)另外初始台地东边也有2个蜂巢。

　　 ⑩静谧公主 / 姫しずか / Silent Princess

　　本作标志性的花，几个重要的升级需要用到。

　　卡卡里克村 / カカリコ村 / Kakariko Village 寺庙后的山坡顶部，就有数朵静谧公主盛开。图中大妖精泉周围同样有数朵。

　　实际上由于游戏中几个重要地点都有长有静谧公主，平时不错过的话应该不会太缺。

　　 ⑪疾走蜥蜴 / ゴーゴートカゲ / Hightail Lizard

　　需求量不算小的素材。作为同样随处可见的素材，更大的问题是平时不容易发现，等突然蹿出来时又往往来不及抓。

　　在很多人一开始摔死的地方(误)精灵之森 / 精霊の森 / Forest of Spirits 就有不少疾走蜥蜴。由于不好观察的原因，最好是先找到一只拍照获取图鉴，再用探索功能进行寻找搜集。

　　——太麻烦了，能不能用钱解决?

　　南卡拉马宿 /ミナッカレ馬宿 / South Akkala Stable 、湖畔马宿 / レイクサイド馬宿 / Lakeside Stable 的商人泰利有售。不过只买一轮是不够用的。

　　 ⑫极限鲈鱼 / マックスバス / Hearty Bass

　　不太多见的鱼，升级需要的数量不少，料理效果也挺不错。

　　塔邦塔大桥马宿 / タバンタ大橋馬宿 / Tabantha Bridge Stable 北边的罗姆山 / ローム山 / Mount Rhoam上有个不起眼的小池子，这个小池子中的鱼基本都是极限鲈鱼。

　　 ⑬潜行鳟鱼 / シノビタマス / Stealthfin Trout

　　难倒很多人的一种鱼。仅出没在海拉尔大森林的水域中，而这里基本是不需要下水的，导致很多人一直没能发现。

　　海拉尔大森林 / ハイラル大森林 / Great Hyrule Forest外围一圈的水域，和森林中的萨莉亚湖 / サリヤ湖 / Lake Saria出没。由于传送点就在湖边，建议反复传送刷取，每次2~4条。

　　 ⑭耐火蝴蝶 / ヒケシアゲハ / Smotherwing Butterfly

　　火山高温地带里随处可见的蝴蝶。然而由于火山地带空气里漂浮着各种燃烧的灰烬，加上耐火蝴蝶本身的保护色，使得耐火蝴蝶并不容易观察到。

　　基本上死亡火山 / デスマウンテン / Death Mountain 高温等级★的区域都有出没。由于难以观察，老样子推荐拍照获取图鉴，再用探索功能寻找。

　　——我知道你很有钱，继续花钱解决对不对?

　　湿地马宿 / 湿原の馬宿 / Wetland Stable、东卡拉马宿 / ヒガッカレ馬宿 / East Akkala Stable 的商人泰利有售，依然是10卢比一只。耐火蝴蝶需求量相对小一点，只靠花卢比搞定也用不了太长时间。

　　 ⑮ 冰蝙蝠翅膀 / アイスキースの羽 / Ice Keese Wing

　　需求量一般，不过大概有些人普通蝙蝠见烦了、冰蝙蝠冰冻效果蛋疼不怎么打，或者比较少探索雪山区域的话会贫缺。

　　冰蝙蝠基本上雪山区域都零零散散分布了很多，不少怪物营地下面都挂着不少，下面介绍一个比较集中的地点。

　　塔邦塔大雪原 / タバンタ大雪原 / Tabantha Tundra 西部有两个骷髅巢穴，其中一个里面就有不少冰蝙蝠挂在下面。

荒野之息》部分紧缺素材高效率收集方法汇总

　　 ⑯ 电蝙蝠翅膀 / エレキースの羽 / Electric Keese Wing

　　和冰蝙蝠翅膀类似，可能很多人更讨厌打电蝙蝠，被电到武器掉到山崖下面捡起来很麻烦什么的...

　　电蝙蝠在很多地方雷雨天都会出没，以及沙漠很多地方出没。不过游戏里有一个电蝙蝠聚集的地方，很方便刷素材。

　　湖畔马宿 / レイクサイド馬宿 / Lakeside Stable 北边的湖泊中有一个叫做ウライト湖 / Riola Spring 的地方，在其北边的山坡上有一条隧道，里面挂有多只电蝙蝠。

　　请做好充分准备再过去，贸然过去可能会被电得动弹不得。

　　 ⑰火蝙蝠翅膀 / ファイアキースの羽 / Fire Keese Wing

　　继续同上，比起冰、电蝙蝠，大家可能更愿意打火蝙蝠吧。火蝙蝠出没在各种高温的地方，相对而言比较常见。

　　奥尔丁之塔 / オルディンの塔 / Eldin Tower 东北角的道路拐了个快180°的地方，是一截隧道。这段隧道里就有大量的火蝙蝠。
